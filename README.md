# AI Resume Builder

AI-powered web application that generates ATS-friendly resumes using Large Language Models.

## Features
- Resume generation using LLMs
- Real-time editing and customization
- Secure authentication
- ATS-friendly output

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Deployment: Vercel, Render
